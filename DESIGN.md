# `/workspace` 暖色文档办理工作台设计规则

本文件是对 `VoltAgent/awesome-design-md` 中 Claude 风格设计说明的项目化摘录。本项目不使用任何 Claude、Anthropic 标志、名称或官方资产，只采用“暖奶油画布、珊瑚主操作、深色产品面板、编辑感排版”的公开视觉语言。

## 设计方向

- 页面定位：跨境卖家的注册办理工作台，重点是“当前要确认什么、有哪些风险、下一步去哪办、材料能否下载”。
- 画布：使用暖奶油 `#faf9f5`，主文字 `#141413`，正文 `#3d3d3a`，边框 `#e6dfd8`。
- 主操作：珊瑚色 `#cc785c`，hover/active 使用 `#a9583e`，按钮圆角 8px，不使用黑色 pill 或大面积薄荷绿。
- 重点承载：用深色面板 `#181715 / #252320` 放置办理摘要、风险核验和最重要官网入口。
- 内容层级：首屏优先当前问题和摘要；历史记录、完整入口、材料长列表默认折叠。

## 组件规则

- 标题使用编辑感 serif fallback：`Georgia, "Times New Roman", serif`；正文和控件继续使用系统 sans。
- 不使用负 letter-spacing，保持项目级可读性规则。
- 卡片以颜色块和 1px hairline 建立层级，少用阴影；卡片圆角约 12px，按钮和输入框圆角约 8px。
- 风险提示使用暖色窄条或小卡，不做大警报区。
- 模板下载区使用“文件抽屉”样式，DOCX / Markdown / HTML 为紧凑按钮。
- 入口链接必须保留真实 `href`、可见 URL、复制按钮和 `data-official-url`。

## 禁用项

- UI 中不出现 `DeepSeek`、`Tavily`、`Brave`、`Claude`、`Anthropic`、`自适应主控`、`动态问诊` 等内部或外部品牌词。
- 不恢复政务蓝、蓝紫渐变、薄荷绿任务台或“航线/舱单/关卡”隐喻。
- 不把免责声明放在首屏最前阻断任务，只保留可展开说明。
- 不新增 Figma、大组件库或品牌资产依赖。
