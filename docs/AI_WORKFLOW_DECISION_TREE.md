# AI 营业执照注册引导工作流 — 完整决策树与问答脚本

> 本文档定义了 AI 对话式引导系统的完整工作流，覆盖「个体工商户」和「企业公司」两种注册类型。
> AI 采用逐步追问方式，根据用户回答动态调整后续问题，最终输出个性化的操作方案。
>
> 生成日期: 2026-05-17 | 数据来源: 4份内部教程文档 + 在线政策调研

---

## 全局工作流概览

```
用户进入系统
  │
  ├─ [阶段0] 欢迎 + 意图识别
  │     └─ 确定注册类型: 个体户 / 企业公司 / 不确定
  │
  ├─ [阶段1] 基础信息收集（两种类型共用）
  │     ├─ 1.1 所在省份/城市
  │     ├─ 1.2 经营行业/方向
  │     └─ 1.3 是否已有部分材料
  │
  ├─ [阶段2] 类型专属信息收集
  │     ├─ [个体户路线] → 阶段2A
  │     └─ [企业公司路线] → 阶段2B
  │
  ├─ [阶段3] 公司名称设计引导
  │
  ├─ [阶段4] 经营范围选择引导
  │
  ├─ [阶段5] 注册地址确认
  │
  ├─ [阶段6] 材料清单生成 + 缺口分析
  │
  ├─ [阶段7] 注册操作指引（线上/线下）
  │
  └─ [阶段8] 注册后续事项引导
```

## 阶段 0：欢迎与意图识别

### AI 开场白
```
你好！我是营业执照注册引导助手。我会一步步帮你搞清楚注册需要准备什么、怎么操作。

先问你一个关键问题：你想注册的是哪种类型？
```

### 问题 Q0.1：注册类型选择
```
你想注册的经营主体类型是？

A. 个体工商户（一个人经营，最简单，适合小生意）
B. 有限责任公司（可以有多个股东，适合团队创业或融资）
C. 不太确定，帮我分析一下
```

#### 分支逻辑：
- **选A** → 进入「个体户路线」，跳转阶段1
- **选B** → 进入「企业公司路线」，跳转阶段1
- **选C** → 触发「类型选择引导子流程」↓

### 子流程：类型选择引导

#### Q0.2: 经营规模判断
```
好的，让我帮你判断。请问：

1. 你计划几个人一起经营？
   A. 就我一个人
   B. 2-3个人合伙
   C. 有明确的投资人/合伙人，需要股权分配

2. 你是否需要对外融资或招投标？
   A. 不需要，就自己做个小生意
   B. 可能需要，未来想做大
   C. 肯定需要，已有投资意向
```

#### AI 判断逻辑：
```
IF (1人经营 AND 不需要融资):
    推荐 = "个体工商户"
    理由 = "注册最简单、成本最低、税务核定征收省事"
    
ELIF (1人经营 AND 可能/肯定需要融资):
    推荐 = "一人有限责任公司"
    理由 = "有限责任保护个人资产，可对外融资"
    风险提示 = "需每年审计，需证明公司财产与个人财产独立"
    
ELIF (多人合伙 AND 不需要融资):
    推荐 = "个体工商户（家庭经营）"
    备选 = "如需明确利润分配，可考虑有限责任公司"
    追问 = "是否需要明确的股权/利润分配比例？"
    注意 = "本系统暂不支持合伙企业和个人独资企业的注册引导，如需注册这两类，建议咨询当地政务大厅"
    
ELIF (多人合伙 AND 需要融资/招投标):
    推荐 = "有限责任公司"
    理由 = "法人主体，有限责任，便于融资和投标"
```

#### Q0.3: 确认推荐
```
根据你的情况，我建议注册 [推荐类型]。

原因：[理由]
[如有风险提示则显示]

你同意这个方向吗？
A. 同意，继续
B. 我想了解更多区别
C. 我想选另一种类型
```

#### 如选B，提供对比说明：
```
【四种类型核心对比】

┌──────────┬─────────────┬───────────────┬──────────────┐
│          │ 个体工商户    │ 个人独资企业    │ 有限责任公司   │
├──────────┼─────────────┼───────────────┼──────────────┤
│ 法律地位  │ 非法人       │ 非法人企业      │ 企业法人      │
│ 责任范围  │ 无限责任     │ 无限责任        │ 有限责任      │
│ 人数要求  │ 1人/家庭    │ 1个自然人       │ 1-50个股东   │
│ 税种     │ 个人所得税   │ 个人所得税      │ 企业所得税+分红税│
│ 设立难度  │ 最简单      │ 简单           │ 较复杂        │
│ 注册资本  │ 无要求      │ 无最低要求      │ 认缴制(5年实缴)│
│ 适合场景  │ 小本生意    │ 个人创业/税筹   │ 团队/融资/长期 │
└──────────┴─────────────┴───────────────┴──────────────┘

特别提醒：
- 个体户承担无限责任（但经营风险通常较低）
- 有限公司的"有限"指股东只在出资范围内担责
- 2024年新《公司法》要求注册资本5年内必须实缴到位
```

## 阶段 1：基础信息收集（共用）

### Q1.1: 所在地区
```
你在哪个省份/城市？（不同地区的政务网站和具体要求可能有差异）

请输入你的省份和城市，例如：四川省成都市
```

**AI 处理逻辑：**
- 记录用户所在地 → 用于后续匹配政务服务网站
- 判断是否为特殊城市（深圳/海南等有独立流程）

### Q1.2: 经营方向
```
你计划做什么行业/生意？简单描述一下即可。

例如：
- "开网店卖服装"
- "做跨境电商"
- "开奶茶店"
- "做软件开发"
- "做自媒体/短视频"
```

**AI 处理逻辑：**
- 识别行业关键词 → 映射到经营范围大类
- 判断是否涉及「许可经营项目」（前置/后置审批）
- 如涉及许可项目，生成提醒

```
IF 行业 in [餐饮, 食品加工, 食品销售]:
    提醒 = "你选择的行业需要办理【食品经营许可证】，这是后置许可，可以先办照再办证"
    
IF 行业 in [教育培训]:
    提醒 = "教育培训可能需要前置许可，建议确认当地政策"
    
IF 行业 in [跨境电商, 进出口]:
    提醒 = "跨境电商需要在经营范围中包含'从事货物及技术的进出口业务'，后续还需办理进出口权备案"
```

### Q1.3: 已有材料盘点
```
你目前已经准备好了哪些材料？（可多选）

□ 身份证（本人）
□ 其他股东/合伙人的身份证
□ 经营地址（已有租房合同或自有房产）
□ 已想好公司名称
□ 已确定经营范围
□ 银行卡（双币信用卡）
□ 以上都还没有准备
```

**AI 处理逻辑：**
- 记录已有材料 → 后续「缺口分析」时跳过已有项
- 如「以上都没有」→ 全流程引导

## 阶段 2A：个体户专属信息收集

### Q2A.1: 经营者信息
```
作为个体工商户的经营者（法人），我需要确认以下信息：

1. 你的姓名？
2. 身份证号码？（用于核名和填报，系统不存储）
3. 联系电话？
4. 是家庭经营还是个人经营？
   A. 个人经营（就我自己）
   B. 家庭经营（家庭成员共同参与）
```

### Q2A.2: 办理方式偏好
```
个体户营业执照有两种办理方式：

A. 线上办理（在省政务服务网上提交，约20分钟，然后去政务大厅打印）
B. 线下办理（直接去政务大厅提交材料，快的话当天出证）

你倾向哪种方式？
```

**AI 处理逻辑：**
- 选A → 后续阶段7提供线上操作步骤
- 选B → 后续阶段7提供线下材料清单和流程

## 阶段 2B：企业公司专属信息收集

### Q2B.1: 公司类型细化
```
有限责任公司有几种细分类型，请确认：

1. 股东人数？
   A. 1个人（一人有限公司）
   B. 2个以上股东

2.（如选B）请列出所有股东姓名：
   格式：姓名1, 姓名2, 姓名3
```

**AI 处理逻辑：**
```
IF 1人:
    公司类型 = "有限责任公司（自然人独资）"
    提醒 = "一人有限公司需每年审计，且需证明公司财产与个人财产独立，否则可能承担连带责任"
    
IF 多人:
    公司类型 = "有限责任公司（自然人投资或控股）"
    继续追问股东比例
```

### Q2B.2: 注册资本
```
注册资本是公司对外承担责任的上限。

重要提醒：2024年新《公司法》要求注册资本必须在5年内实缴到位（真金白银打到公司账户）。

请问你计划填多少注册资本？

参考建议：
- 咨询/服务类：10-50万
- 科技/软件：50-100万
- 贸易/电商：50-200万
- 不确定，帮我建议
```

#### 如选"帮我建议"，追问：
```
为了给你建议，请告诉我：

1. 你5年内能实际出资多少钱？
   A. 10万以内
   B. 10-50万
   C. 50-100万
   D. 100万以上

2. 是否需要参与招投标或面向大客户？
   A. 不需要
   B. 需要（某些项目有注册资本门槛要求）

3. 是否计划冠省名？（如"广东XX公司"而非"深圳市XX公司"）
   A. 不需要（冠市名即可）
   B. 想要冠省名（需注册资本500万以上）
```

**AI 建议逻辑：**
```
建议金额 = min(用户实际出资能力, 行业合理范围)

IF 需要招投标:
    建议金额 = max(建议金额, 100万)
    
IF 冠省名:
    建议金额 = max(建议金额, 500万)
    备注 = "冠省名注册资本要求各省不同（通常500万以上），请以当地规定为准"
    
风险提示: f"注册资本{建议金额}万意味着你需要在5年内将{建议金额}万元打入公司对公账户。
如果公司经营失败负债，股东最多在{建议金额}万范围内承担责任。"
```

### Q2B.3: 股东出资比例（多人公司）
```
请确认各股东的出资比例：

格式示例：
- 张三: 60%
- 李四: 30%
- 王五: 10%

（所有股东比例之和必须等于100%）
```

### Q2B.4: 人员信息
```
公司需要确定以下人员（一人可兼多职，但法定代表人和监事不能是同一人）：

1. 法定代表人：谁担任？（通常是最大股东或实际负责人）
   姓名：___

2. 执行董事/董事长：（可与法定代表人同一人）
   姓名：___

3. 监事：（不能与法定代表人、董事、经理是同一人）
   姓名：___
   
   ⚠️ 一人有限公司也必须设监事，可以找一个信任的人（家人/朋友）挂名

4. 财务负责人：
   姓名：___
   联系电话：___

5. 联络员：（日常联系人，谁办理就填谁）
   姓名：___
   联系电话：___
   邮箱：___
```

**AI 校验逻辑：**
```
IF 法定代表人 == 监事:
    错误提示 = "法定代表人和监事不能是同一人，请更换监事人选"
    
IF 一人有限公司 AND 未填监事:
    提醒 = "一人有限公司也必须设一名监事，可以让家人或朋友担任"
```

### Q2B.5: 出资时间
```
各股东的出资时间（即注册资本什么时候实际到位）：

建议：可以设置为公司成立后的某个时间，但不能超过5年。
例如：2029年12月31日（距今约3.5年）

你计划设置的出资截止日期是？
A. 公司成立后1年内
B. 公司成立后3年内
C. 公司成立后5年内（法定最长期限）
D. 自定义日期
```

### Q2B.6: 办理方式偏好
```
公司注册也有两种办理方式：

A. 线上办理（在省政务服务网上提交，全程电子签名，适合全体股东都在本省的情况）
B. 线下办理（到政务大厅窗口提交，适合有外省股东或材料复杂的情况）

你倾向哪种方式？
```

**AI 处理逻辑：**
- 选A → 后续阶段7提供7C线上操作步骤
- 选B → 后续阶段7提供7D线下操作步骤
- 如有外省法人股东或15位身份证：自动建议线下办理

## 阶段 3：公司名称设计引导

### Q3.1: 名称规则说明 + 收集意向
```
营业执照名称由四部分组成：

【行政区划】+【字号】+【行业】+【组织形式】

举例：
- 个体户：成都市｜锦绣｜服装｜店
- 公司：  深圳市｜神舟｜网络科技｜有限公司

其中「字号」是你自己起的名字，其他部分有固定格式。

现在请告诉我：
1. 你想的字号（公司名称中独特的部分）？请提供2-3个备选
   例如：第一选择"星辰"，备选"天行""远航"

2. 行业表述你想用什么？（需与经营范围匹配）
   常见选择：网络科技/电子商务/贸易/商贸/信息技术/文化传媒
```

### Q3.2: 名称合规检查
```
AI 内部检查逻辑（自动执行，不展示给用户）：

CHECK_LIST:
- [ ] 字号是否 >= 2个汉字
- [ ] 是否包含禁用词（国家名、政党名、损害公共利益）
- [ ] 是否包含数字、字母、外文
- [ ] 是否与常见知名企业名称相同/近似
- [ ] 行业表述是否与经营范围匹配
```

**检查通过后：**
```
你的名称方案：

✅ 第一选择：[城市] + [字号1] + [行业] + [组织形式]
   完整名称：深圳市星辰网络科技有限公司

✅ 备选1：[城市] + [字号2] + [行业] + [组织形式]
✅ 备选2：[城市] + [字号3] + [行业] + [组织形式]

⚠️ 注意事项：
- 名称需要在系统中查重，有可能被占用
- 同一行业内不能有相同或谐音字号
- 建议至少准备3个备选名称

这些名称你满意吗？我们继续下一步。
```

**检查不通过的反馈示例：**
```
❌ "华为"这个字号不可用，原因：与知名企业字号相同
❌ "中国星辰"不可用，原因：使用"中国"字样需国务院批准
❌ "ABC科技"不可用，原因：字号不能包含字母

请重新提供字号。
```

## 阶段 4：经营范围选择引导

```
  ✓ [根据行业搭配推荐]
```

### 8.1 刻章详情（如选A）
```
【刻章办理指引】

必须刻的章（至少3枚）：
1. 公章 — 对外签约、开证明必用
2. 财务章 — 银行业务、报税必用
3. 法人私章 — 支票、汇款必用

建议加刻：
4. 发票专用章 — 开发票必用
5. 合同章 — 签合同用（可用公章代替）

⚠️ 必须去公安局备案的刻章点！非备案刻章点刻的章没有法律效力。

费用：200-600元/套（部分地区政府免费提供首套）
时间：1-2个工作日

如何找备案刻章点：搜索"[你的城市] 公安备案刻章"或咨询政务大厅
```

### 8.2 银行开户详情（如选B）
```
【银行开户（对公基本户）指引】

流程：
1. 电话/线上预约开户时间（必须预约）
2. 法人本人携带材料到银行办理
3. 银行审核签约
4. 1-5个工作日开通

所需材料：
□ 营业执照正副本原件
□ 法人身份证原件
□ 公章
□ 财务章
□ 法人私章
□ 公司章程（企业需要）

⚠️ 注意事项：
- 法人必须本人到场！不能代办
- 部分银行会上门拍照核实经营地址
- 建议选择离经营地址近的银行

费用：年费约100-2000元（各银行不同，小微企业可享减免）
建议：找几家银行对比年费和服务
```

### 8.3 税务登记详情（如选C）
```
【税务登记/报到指引】

⏰ 法定期限：领取营业执照后30天内必须办理！

核心事项：
1. 确认纳税人身份
   - 小规模纳税人：年销售额≤500万，增值税3%（优惠税率以当年财税政策为准）
   - 一般纳税人：年销售额>500万，增值税6%-13%
   
2. 核定税种
   - 增值税
   - 企业所得税（公司）/ 个人所得税（个体户）
   - 附加税

3. 签订银税三方协议（需先有银行对公账户）

所需材料：
□ 营业执照副本
□ 法人身份证
□ 公章
□ 银行开户许可证（签三方协议用）
□ 公司章程（企业）

办理地点：
- 线上：当地电子税务局网站
- 线下：主管税务局办税大厅

后续义务：
- 自登记次月起按期申报（月报或季报）
- 无论是否有收入，都必须按时申报（零申报）
- 代理记账费参考：小规模200-400元/月，一般纳税人400-800元/月
```
